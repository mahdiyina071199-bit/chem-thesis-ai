# CHEM-THESIS AI

CHEM-THESIS AI adalah web app sederhana yang berisi prompt sistem dan menu kerja untuk membantu pengembangan tesis pendidikan kimia.

## Fokus Penelitian

Penerapan Model Project Based Learning melalui Pembuatan Pasta Gigi dari Ekstrak Bahan Alam Lokal dan VCO pada Materi Koloid untuk Meningkatkan Literasi Halal dan Kemampuan Berpikir Kritis Siswa.

## Fungsi Utama

Aplikasi ini membantu peneliti dalam:

1. Menyusun BAB I.
2. Menyusun BAB II.
3. Menyusun BAB III.
4. Mengembangkan instrumen penelitian.
5. Menganalisis data penelitian.
6. Menyusun pembahasan.
7. Menyiapkan simulasi seminar proposal atau sidang tesis.

## File Utama

- `index.html`
- `style.css`
- `script.js`
- `README.md`

## Cara Menjalankan

1. Download atau clone repository.
2. Buka file `index.html` di browser.
3. Gunakan menu yang tersedia.
4. Salin prompt sistem untuk dimasukkan ke AI Studio atau chatbot AI lain.

## Cara Deploy ke GitHub Pages

1. Upload semua file ke repository GitHub.
2. Masuk ke menu `Settings`.
3. Pilih `Pages`.
4. Pada bagian `Source`, pilih branch `main`.
5. Pilih folder `/root`.
6. Klik `Save`.
7. Tunggu sampai link GitHub Pages aktif.

## Lisensi

Proyek ini dibuat untuk kebutuhan akademik dan pengembangan tesis pendidikan kimia.
